# BioFlow — Monitoramento de Biorreatores

Sistema de gestão e monitoramento de 6 biorreatores industriais (R-01 a R-06).

## Funcionalidades

- Dashboard com 6 cards de biorreatores coloridos por status
- Cronômetro em tempo real por fase
- Gaveta lateral para troca de fases com registro de log automático
- Rastreamento de operador por fase
- Indicador de próxima etapa sugerida
- Página de histórico com lotes finalizados
- Exportação de lotes para PDF
- Log de atividades completo

## Processos suportados

- Ocioso, Desmontagem, Limpeza, Esterilização c/ Água, CIP, Esterilização a Seco, Esteril. Meio de Cultura, Em Processo

## Setup

```bash
npm install
npm run dev
```
